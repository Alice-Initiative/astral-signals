Quickstart Flow

Start simple, then go deeper only when you need to

Quickstart mirrors a guided song desk: describe the song, choose the feel, set the language and singer, then render. Studio unlocks the full lab.

1. Start with the idea

Pick a preset or write the core brief, then choose genre, mood, language, and duration.

2. Let Astral draft

Ollama writes the plan, then you can stay on ACE-Step for full songs or switch to MusicGen for faster instrumental or wordless passes.

3. Go deeper only if needed

Switch to Studio when you want manual lyrics, multi-singer routing, Alice Lab, Voicebox cues, or stem editing.
